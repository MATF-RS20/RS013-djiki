Input: An array and searched value x
Output: Index of element if it is found, -1 instead

1. Compare x with the middle element
2. If x matches with middle element
3. Then return the middle index
4. Else if x is greater than the middle element
5. Then recursively continue searching in right half of the array
6. Else (x is smaller)
7. Then recursively continue searching in left half of the array
8. When there are no more elements to examine return -1
