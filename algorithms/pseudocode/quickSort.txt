Input: An array, variable low (the starting index), variable high (the ending index)
Output: Sorted array

1. If low < high
2. Then set variable pi to the result of calling partition function: partition(array, low, high)
3. Recursively call algorithm for: array, low, pi - 1
4. Recursively call algorithm for: array, pi + 1, high

Paritition(array, low, high):
1. Set pivot to array[high]
2. Set i to low - 1
3. For j = low to j <= high -1 do
4. If array[j] < pivot:
5. Increment variable i and swap array[i] and array[j]
6. End for loop
7. Swap array[i+1] and array[high]
8. Return i+1
